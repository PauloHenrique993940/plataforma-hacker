Uma plataforma completa e profissional para treinar habilidades de penetration testing, desenvolvida com **React 18**, **Vite**, **TypeScript** e design **hacker** premium.

# 2. Instalar dependências (já feito)

##  Como Usar

### Login
1. Ao abrir a plataforma, clique em "Entrar na Plataforma"
2. Você será autenticado como "Hacker"

### Navegar
- Use o **menu lateral** para acessar diferentes seções
- Clique no **ícone de menu** no mobile para abrir/fechar sidebar
- Use o **header** para ver seus stats (Level, Points, Rank)

### Completar Missões
1. Acesse "Missões"
2. Filtre por status (Todas, Pendentes, Concluídas)
3. Clique em uma missão para ver detalhes
4. Complete o objetivo e clique "Verificar Flag"

### Iniciar Labs
1. Acesse "Laboratórios"
2. Selecione um lab
3. Clique "Iniciar Lab" para começar
4. Use o IP fornecido para conectar
5. Complete o objetivo

### Usar Ferramentas
1. Acesse "Ferramentas"
2. Navegue por categoria
3. Clique "Iniciar" para ativar uma ferramenta
4. Consulte o "Uso Padrão" para exemplos
